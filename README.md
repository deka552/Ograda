# Ograda

Автоматизированный измерительный стенд на базе ESP8266 для мониторинга напряжения с уведомлениями через Telegram.

## Что делает

Прошивка считывает аналоговый сигнал с АЦП (вход A0), преобразует его в значение напряжения (0–5 В) и выводит в Serial. При падении напряжения ниже порогового значения (2 В) отправляет уведомление в Telegram-бот.

## Железо

- ESP8266 (NodeMCU / Wemos D1 Mini)
- Делитель напряжения или датчик на вход A0
- Питание 5 В (USB)

## Схема подключения

```
Измеряемая цепь --> Делитель напряжения --> A0 (ESP8266)
```

> АЦП на ESP8266 принимает 0–1 В, при использовании встроенного делителя на NodeMCU — 0–3.3 В. Убедитесь, что входное напряжение не превышает допустимый диапазон.

## Сборка и прошивка

**Зависимости:**
- PlatformIO или Arduino IDE
- Библиотека [FastBot2](https://github.com/GyverLibs/FastBot2)

**Настройка:**

1. Скопируйте `config.h.example` в `config.h`
2. Заполните параметры Wi-Fi и Telegram-бота
3. Скомпилируйте и залейте прошивку

```bash
cp config.h.example config.h
# отредактируйте config.h — укажите SSID, пароль, токен бота и chat ID
```

## Используемые интерфейсы и технологии

- АЦП — считывание аналогового сигнала
- Wi-Fi — подключение к сети и отправка данных
- Telegram Bot API (через FastBot2) — удалённые уведомления
- UART (Serial) — отладочный вывод

## Возможные доработки

- Добавить измерение тока через INA219 (I2C)
- Подключить датчики температуры/влажности (DHT22, MAX6675)
- Логирование данных на SD-карту или отправка на сервер
- Веб-интерфейс для просмотра показаний в реальном времени
